# BigBonusBlitz (BBB) ゲーム仕様書

このドキュメントは、スロットゲーム「BigBonusBlitz (BBB)」のゲーム構成、確率、および各種設定をまとめた資料です。

## 1. 基本仕様
- **タイプ**: Aタイプ相当（ボーナス主体）＋ 内部モード管理
- **リール**: 3リール（各20コマ）
- **配当（通常時）**:
  - **BIG BONUS (BB)**: 300枚獲得で終了
  - **REGULAR BONUS (RB)**: 90枚獲得で終了
  - **STAR（ベル相当）**: 8枚
  - **WATERMELON（スイカ）**: 6枚
  - **CHERRY（チェリー）**: 4枚
  - **REPLAY（リプレイ）**: 再遊技（3枚相当）
- **ボーナス中配当**:
  - **BB中**: STAR 15枚, WATERMELON 6枚, CHERRY 4枚
  - **RB中**: STAR 15枚, WATERMELON 6枚, CHERRY 4枚

## 2. 状態・モード構成
ゲームは以下の状態（モード）で管理されています。

| モード名 | 概要・特徴 |
| :--- | :--- |
| **モードA (A)** | 基本となる通常モード。 |
| **モードB (B)** | 通常モードの一つ。 |
| **モードC (C)** | 通常モードの一つ。 |
| **Tier2** | チャンスゾーン・高確状態（最大20G継続）。STAR（ベル）確率が劇的に上昇し、メダルを増やしながらボーナスを狙える状態。 |
| **BB中 (BB)** | BIG BONUS消化中の状態。ハズレ確率が下がり、STAR確率が大幅に上昇。 |
| **RB中 (RB)** | REGULAR BONUS消化中の状態。特定役の確率が変化。 |
